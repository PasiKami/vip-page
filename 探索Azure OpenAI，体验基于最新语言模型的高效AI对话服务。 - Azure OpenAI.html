代码编程

无论是大神还是小白都能写出标准的代码，写出来就能运行，人人都能成为编程高手。

帮我写一段Python代码，功能是获取字符串”Hello World!“的hash值。

当使用哈希函数加密字符串时，可以使用Python内置的哈希函数hashlib。下面是一个获取字符串“Hello World!”的哈希值的Python代码：
import hashlib
str = “Hello World!”
hash_object = hashlib.md5(str.encode())
hash_value = hash_object.hexdigest()
print(“字符串的哈希值为：”, hash_value)
